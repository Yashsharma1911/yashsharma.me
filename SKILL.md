## Quick orientation

- The aesthetic: editorial, type-led, monochrome with one oxblood accent.
- Tokens live in `colors_and_type.css` (CSS custom properties) — import this file and you have everything. The shipped palette is **Palette A — Oxblood on warm paper**: paper `#F4F1EA`, ink `#1A1714`, accent `#7A1C1C`.
- Type stack: Instrument Serif (display) + Geist (body) + JetBrains Mono (numerals/dates/venues). Loaded from Google Fonts.
- Component patterns: see `preview/` for the spec cards (organized into Type / Colors / Spacing / Components / Brand groups).
- **The finalized homepage:** `palette-iterations/index-A-oxblood.html` + `palette-iterations/page.jsx` — single React + Tailwind document with all sections (Nav, Hero, Logo wall, Credibility strip, About, Signature talks, Pull-quote, Photo marquee, Content index, Open source, second Pull-quote, Social, Press, Footer). Do not edit this file.
- Earlier modular breakdown lives in `ui_kits/personal-site/` for reference only — the live build is in `palette-iterations/`.

## Hard rules
- No gradients. No tinted section backgrounds beyond `--paper-deep` for the credibility and pull-quote lifts.
- Accent (oxblood `#7A1C1C`) appears in <5% of pixels — credibility numerals, hover states, the italic thesis line under the hero, social-card metric primary, and focus rings only.
- Sentence case for headings; mono labels in lowercase.
- No emoji anywhere. No exclamation marks.
- Borders are 1px hairlines in `--rule` (`#D8D2C5`). Corner radii max out at 4px.
- One easing curve site-wide: `cubic-bezier(0.2, 0, 0, 1)`. Three durations: 120 / 240 / 480 ms.
- The photo marquee is the *only* continuous animation on the page.
- Photographic imagery only. No illustration, no SVG art beyond logos.
